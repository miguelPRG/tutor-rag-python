# Variáveis em Python

Uma variável é como uma caixa onde guardas um valor para usares mais tarde. Em Python, crias uma variável atribuindo-lhe um nome e um valor.

## Como criar:
Usa o símbolo `=` (operador de atribuição).
Exemplo: `idade = 18`

## Regras de nomes:
- Devem começar com uma letra ou underscore (`_`).
- Não podem começar com números.
- Só podem conter caracteres alfanuméricos e underscores (A-z, 0-9, e _).
- São sensíveis a maiúsculas e minúsculas (`idade` é diferente de `Idade`).

## Erro Comum:
Tentar usar uma variável antes de a definir resulta num `NameError`.
Exemplo:
```python
print(pontos) # Erro se 'pontos' não foi criado antes!
```
