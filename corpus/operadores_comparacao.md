# Operadores de Comparação

Estes operadores são usados para comparar dois valores e devolvem `True` ou `False`.

## Os Operadores:
- `==` : Igual a (Note os dois iguais!)
- `!=` : Diferente de
- `>`  : Maior que
- `<`  : Menor que
- `>=` : Maior ou igual a
- `<=` : Menor ou igual a

## Diferença entre = e ==:
- `=` é para **atribuir** (ex: `x = 5` guarda 5 em x).
- `==` é para **comparar** (ex: `x == 5` pergunta se x é 5).

## Exemplo:
```python
x = 10
y = 20
print(x < y) # True
```
